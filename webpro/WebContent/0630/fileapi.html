1. file API 읽기, 첨부

- 첨부파일 값 가져오기 : files
- File 객체를 리턴한다.
- File 객체에서 정보를 얻는다. - 이름(name), 수정한 날짜(lastModifiedDate), 크기(size), 유형(type)

- 첨부된 파일 내용 읽기
- FileReader 객체
- 메서드
- readAsText(file객체, encoding)
- readAsDataURL(file객체)
- 속성
- result : 읽은 파일의 내용을 저장하는 속성
- error : 에러 발생 시 에러 정보를 저장하는 속성
