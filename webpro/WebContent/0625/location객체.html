1. Location객체

- href : 전체 URL
- hostname : 서버이름, URL중에서 hostname을 반환
- host : URL중에서 hostname과 port를 반환 (포트 : 데이터 출입구 번호)
- pathname : URL중에서 경로(path)를 반환, port이후의 정보
- protocol : URL중에서 protocol을 반환, 규약, 서비스 종류
