label : 문자, 인라인 요소
	=> 크기 조절이 안된다
	=> display
	
display : inline, block, inline-block
	- inline-block : 인라인이면서 블럭요소의 속성을 동시에 갖는다
	- 즉, 줄은 바뀌지 않으면서 크기 조절이 가능하다

입력요소(input요소, textarea)의 값을 가져오거나 새로운 값으로 설정 - value
입력요소가 아닌 다른 요소의 값을 가져오거나 새로운 값으로 설정 - innerHTML, textContent

- innerHTML과 textContent의 차이점 
: textContent는 태그가 실행되는 것이 아니라 그대로 출력되고
  innerHTML은 태그가 실행되어 출력화면에 태그가 보이지 않는다.

textarea : 여러 줄 입력. 출력시에도 줄바꿔 줄이 바뀌도록 한다
varea의 값을 엔터처리 (입력 시  엔터 : \r\n -> <br>)
replace(/old/g, new)
varea = varea.replace(/\r/g, "").replace(/\n/g, "<br>");

두 개의 span요소를 세로 맞추기 : vertical-align : top;
