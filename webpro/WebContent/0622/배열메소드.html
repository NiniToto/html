1. 배열

- a = [11, 100, 180, 53, 22, 230, 450]
- Reverse() : 배열을 거꾸로 출력
- Sort() : 100, 11 ,180, 22, 230, 450, 53
- Sort(function(a,b)){} : 오름차순
- Sort(function(a,b)){} : 내림차순

2. 배열

- a2 = ["사과", "포도", "수박", "참외", "파인애플", "망고", "딸기", "자몽"]
- Slice(start, end) : start ~ end-1 까지의 추출
- Splice(idx, n, expr1, ...) : 수정 - 삽입/추가, 삭제, 삭제후 추가
- Splice(3, 0, "메론") : 3번째 위치에 메론을 삽입
- Splice(3, 1, "메론", "자두") : 3번째 인덱스에 있던 기존 데이터를 1개 지우고 메론, 자두를 삽입
- Splice(2,2) : 2번째 위치에서 2개의 요소를 삭제한다.

3. 배열

- a = [11, 100, 180, 53, 22, 230, 450]
- a2 = ["사과", "포도", "수박", "참외", "파인애플", "망고", "딸기", "자몽"]
- a.concat(a2) : a와 a2가 합쳐져서 새로운 배열을 만든다
- a.join() : a배열을 하나의 문자열로 변환한다.
- a2.join() : a2배열을 하나의 문자열로 변환한다.
