3. html/text

- 모든 p태그를 대상으로 실행할떄는 반복문을 사용
- 반복문 안에서 this로 각 p태그를 접근한다
- $(요소선택).each(function(매개변수1, 매개변수2){...});
- $.each("요소선택", function(매개변수1, 매개변수2){...});

- $("#result3").html(값) : #result3에 출력
- $("#result3").text(값) : 만약에 값에 html태그가 포함되어 있어도 문자화한다.
